## Conventions

`#` is a local index (`D-##`, no governed prefix). `Type` Ōłł Feature-deferral ┬Ę Manual-for-now ┬Ę Tech-debt ┬Ę Ops-config. `Status` Ōłł Open ┬Ę In progress ┬Ę Done ┬Ę Won't-do. A deferred item is activated when its Trigger is met; activation creates a backlog item and (if it changes a settled decision) an ADR.
